AI Menu Parser — Add Your API Key

The "Parse Menu" button uses Claude's API to fetch today's PDF and extract the lunch items. To enable it, add your Anthropic API key as a GitHub Pages environment variable or paste it directly into the source.

Find this line in the JS and replace the placeholder:

// In the fetchMenu() function: ``` const ANTHROPIC_KEY = “sk-ant-YOUR-KEY-HERE”; // Get one free at console.anthropic.com
⚠️ If hosting publicly on GitHub Pages, use a backend proxy (like a free Cloudflare Worker) to keep your API key private. See Step 3.

Morning Notification — GitHub Actions + SMS

▼

Add this workflow file to your repo at .github/workflows/daily-menu.yml to get a text every school morning at 7am with today's menu.

name: Daily Lunch Menu ``` on: schedule: # 7am ET Mon-Fri (12:00 UTC) - cron: ‘0 12 * * 1-5’ workflow_dispatch: # manual trigger jobs: send-menu: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 ``` - name: Parse menu & send SMS env: ANTHROPIC_KEY: ${{ secrets.ANTHROPIC_KEY }} TWILIO_SID: ${{ secrets.TWILIO_SID }} TWILIO_TOKEN: ${{ secrets.TWILIO_TOKEN }} TO_PHONE: ${{ secrets.TO_PHONE }} FROM_PHONE: ${{ secrets.FROM_PHONE }} run: node scripts/send-menu.js

The script (scripts/send-menu.js) downloads today's PDF, sends it to Claude, and texts the result via Twilio (free trial = 1000 free SMS).

💡 Store all keys in GitHub repo → Settings → Secrets. Never commit them to code.
4

Node Script — send-menu.js

▼

Create scripts/send-menu.js in your repo:

const https = require('https'); ``` const Anthropic = require(’@anthropic-ai/sdk’); const twilio = require(‘twilio’); // Build current month’s PDF URL function getMenuUrl(school) { const d = new Date(); const m = d.toLocaleString(‘en-US’,{month:‘long’}); const y = d.getFullYear(); const base = `https://www.orcsd.org/departments/child-nutrition/files/documents/2025-2026 Lunch Menus/${m} ${y} Lunch Menus/`; const files = { mw: `Mast Way and Moharimet Menus - ${m} ${y}.pdf`, ms: `Oyster River Middle School Lunch Menu - ${m} ${y}.pdf`, hs: `Oyster River High School Lunch Menu - ${m} ${y}.pdf` }; return base + encodeURIComponent(files[school]); } async function main() { // Fetch PDF as base64 const res = await fetch(getMenuUrl(‘mw’)); const buf = await res.arrayBuffer(); const pdf64 = Buffer.from(buf).toString(‘base64’); // Ask Claude what’s for lunch TODAY const ai = new Anthropic({apiKey: process.env.ANTHROPIC_KEY}); const today = new Date().toLocaleDateString(‘en-US’,{weekday:‘long’,month:‘long’,day:‘numeric’}); const msg = await ai.messages.create({ model: ‘claude-opus-4-6’, max_tokens: 300, messages: [{ role: ‘user’, content: [{ type: ‘document’, source: {type:‘base64’, media_type:‘application/pdf’, data: pdf64} },{ type: ‘text’, text: `Today is ${today}. List ONLY today's lunch items from Mast Way, one per line. Be brief.` }] }] }); const menu = msg.content[0].text; const body = `🍽 Mast Way Lunch (${today.split(',')[0]}):\n${menu}`; // Send SMS via Twilio const client = twilio(process.env.TWILIO_SID, process.env.TWILIO_TOKEN); await client.messages.create({ body, to: process.env.TO_PHONE, from: process.env.FROM_PHONE }); console.log(‘✓ Menu sent:’, body); } main().catch(console.error);
📦 Run npm init -y && npm install @anthropic-ai/sdk twilio in your repo root, then commit package.json.
